# FEATBIT EXPERIMENTALS

Experimental and tutorial projects that allow customers to apply required tasks using official approaches, but are not officially included in FeatBit Dashboard.

# Project Structure

- **Dashboard (`dashboard`)**: The front-end app for FeatBit Experimentals Dashboard. **All UIs for interacting with experimental features are provided in this project**.
- `delete-unused-events-data`: A project that demonstrates how to delete unused events data from the database to optimize storage and performance.
